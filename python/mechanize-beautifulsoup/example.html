Introducing Datomic to C#.NET

24 Feb 2013 at 03:08

kacey

Projects

No Comments

There's a new project brewing. A recent conversation has started me on an implementation of Datomic via EDN parsing with C#.NET.

Datomic is a modern data management system that works to seamlessly integrate into a programming implementation, serializing and de-serializing data into entities with related facts, which are stored with time elements, allowing one to parse prior states.

Getting into the meat of the implementation has been a challenge. Datomic relies on EDN as a standard for serialization, and it was designed with functional languages in mind. Type safety in C style languages is a huge hurdle to overcome in transforming the data between the systems, giving that seamless feel.

I'll post updates as I go along.
